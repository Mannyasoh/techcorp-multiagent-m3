IT SUPPORT GUIDE

GETTING STARTED

New Employee Setup
All new employees receive a company laptop within 2 business days of their start date. Standard configurations include:
- MacBook Pro 14" (Development) or MacBook Air 13" (General)
- 16GB RAM minimum, 32GB for development roles
- 512GB SSD storage standard
- Pre-installed software: Office 365, Slack, VPN client, security tools

Initial Setup Process
1. Receive laptop from IT with initial admin account
2. Complete security training within first week
3. Set up MFA for all company accounts
4. Install required software from company portal
5. Join domain and sync with Active Directory
6. Complete backup configuration with Time Machine

Account Management
Company email accounts are created automatically upon hiring. Username format: firstname.lastname@techcorp.com
Default passwords must be changed within 24 hours. Password requirements:
- Minimum 12 characters
- Mix of uppercase, lowercase, numbers, symbols
- No dictionary words or personal information
- Cannot reuse last 5 passwords

HARDWARE SUPPORT

Laptop Issues
Common troubleshooting steps:
1. Restart computer - resolves 70% of issues
2. Check for software updates
3. Clear browser cache and cookies
4. Run disk utility to check for errors
5. Reset PRAM/NVRAM if performance issues persist

For hardware failures, contact IT helpdesk immediately. Replacement laptops available within 4 hours for critical issues. All data should be backed up to corporate cloud storage.

Mobile Devices

- iPhone 14 for general business use
- Monthly plan includes unlimited data and international roaming
- MDM enrollment required for security compliance

NETWORK AND CONNECTIVITY

WiFi Access
Corporate WiFi networks:
- TechCorp-Secure: WPA3 enterprise authentication
- TechCorp-Guest: Temporary access for visitors
- TechCorp-Devices: IoT and smart devices

VPN Configuration
All remote work requires VPN connection:
- FortiClient VPN client pre-installed
- Two-factor authentication required
- Split tunneling enabled for performance
- Auto-connect when on untrusted networks
- Kill switch prevents unencrypted traffic

Network Troubleshooting
Connection issues checklist:
1. Check cable connections
2. Restart network adapter
3. Flush DNS cache
4. Reset network settings
5. Contact ISP if home internet issues
6. Use mobile hotspot as backup

SECURITY POLICIES

Data Protection
All company data must be stored in approved locations:
- OneDrive Business for personal files
- SharePoint for team collaboration
- AWS S3 for application data
- No personal cloud storage permitted

USB devices require IT approval and encryption
Email attachments scanned automatically
Web browsing filtered for malicious content
Software installation requires admin approval

Multi-Factor Authentication
Required for all company accounts:
- Office 365 and email access
- GitHub and development tools
- AWS and cloud services
- VPN and remote access
- Administrative systems

MFA methods supported:
- Microsoft Authenticator (preferred)
- SMS text messages
- Hardware security keys
- Phone call verification

Incident Response
Security incidents must be reported immediately:
1. Disconnect affected device from network
2. Contact IT security team: security@techcorp.com
3. Document incident details and timeline
4. Preserve evidence for investigation
5. Follow remediation instructions

Common security incidents:
- Phishing email received
- Malware detection
- Lost or stolen device
- Unauthorized access attempt
- Data breach suspected
